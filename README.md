# Projeto: Clusterização de Pinguins com K-means

Este projeto usa o algoritmo K-means para segmentar pinguins com base em suas características físicas.
O dataset utilizado é o 'penguins' do pacote seaborn, que inclui medidas como:

- Comprimento e profundidade do bico
- Comprimento da barbatana
- Massa corporal
- Espécie (Adelie, Chinstrap, Gentoo)

## Etapas realizadas:
1. Carregamento e limpeza dos dados (remoção de nulos e colunas categóricas)
2. Visualização com pairplot para observar agrupamentos
3. Padronização dos dados com StandardScaler
4. Aplicação do algoritmo K-means com 3 clusters
5. Visualização dos clusters e centroides com matplotlib
6. Discussão de utilidades do algoritmo de clusterização

## Utilidades práticas da clusterização:
1. Segmentação de clientes com base em comportamento de compra
2. Organização automática de grandes volumes de texto por similaridade
3. Detecção de comportamentos anômalos (ex: fraudes)

Projeto feito com Google Colab | Linguagem: Python | Bibliotecas: seaborn, pandas, matplotlib, scikit-learn
