# Projeto-Clusteriza-o-de-Pinguins-com-K-means
Este projeto usa o algoritmo K-means para segmentar pinguins com base em suas características físicas. O dataset utilizado é o 'penguins' do pacote seaborn, que inclui medidas como:  - Comprimento e profundidade do bico - Comprimento da barbatana - Massa corporal - Espécie (Adelie, Chinstrap, Gentoo)
